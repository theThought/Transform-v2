# Color Primitives

The foundational color palette of the design system. These raw values define every available color and serve exclusively as the source for semantic color tokens. Primitive colors should never be applied directly to components — always reference them through semantic tokens.

---

The core colors that express the primary visual identity of the brand.

## System — Success

Colors reserved for communicating specific system states: success, error, warning, and information. Each state includes three tonal variations to support background, fill, and text applications consistently.
