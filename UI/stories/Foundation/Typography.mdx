# Typography

Defines the typeface pairings, font size scale, and default line height used across the design system. These tokens establish the typographic foundation for all text styles — for both left-to-right (LTR) and right-to-left (RTL) reading directions.

## Line Height

The default line height applied across all text styles. Expressed as a unitless multiplier relative to the font size.
