# Typography

Defines the typeface pairings, font size scale, and default line height used across the design
system. These tokens establish the typographic foundation for all text styles — for both
left-to-right (LTR) and right-to-left (RTL) reading directions.

---

The type scale defining all available text sizes.

## Line Height

The default line height applied across all text styles. Expressed as a unitless multiplier
relative to the font size.

  type="spacing"
  showDescription={true}
  tokens={[
    { name: '--line-height', value: '1.5', description: '150% — Unitless ratio multiplied by the element\'s font size' },
  ]}
