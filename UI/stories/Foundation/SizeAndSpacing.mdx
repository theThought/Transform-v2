# Size & Spacing

Spacing tokens define the standard distances used for padding, margin, and gap across all components
and layouts. Built on a consistent scale, they remove guesswork from spacing decisions and ensure
visual rhythm and alignment throughout the interface.

---

## Layout

Structural tokens that define the overall page layout. These values set the maximum content width,
page margins, column gaps, and standard input widths.
